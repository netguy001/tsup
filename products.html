LockBox

In Development

A secure vault system for managing passwords, API keys, tokens, secrets, files, folders, and sensitive configuration data. Uses specialized high-level encryption that protects against unauthorized access, including malware.

What it does

LockBox provides centralized, encrypted storage for all types of credentials, files, folders, and sensitive data. Anything stored in LockBox is encrypted with specialized high-level cryptography that only LockBox can decrypt. Even if malware accesses files inside LockBox, they remain completely secure and unreadable.

How it works

Everything stored in LockBox—passwords, files, folders, documents—is encrypted using specialized cryptographic algorithms before being written to storage. Data remains encrypted at rest and is only decrypted at runtime when an authenticated and authorized request is made through LockBox itself. No plain-text secrets or files are ever stored on disk.

Key capabilities
  • Encrypts and secures passwords, API keys, SSH keys, certificates, files, and folders
  • Specialized high-level encryption that only LockBox can decrypt
  • Protection against malware—encrypted files remain secure even if accessed
  • Granular access controls with role-based permissions
  • Auto-locking and clipboard protection for enhanced security
  • Offline-first architecture with local data sovereignty

Scrapify

Limited Access

AI-powered web intelligence platform for structured data extraction, research automation, and multi-source analysis. Built for teams that need accurate, compliant data collection at scale.

What it does

Scrapify collects and analyzes information from websites and digital sources. It extracts text content, links, media references, metadata, and both structured and unstructured data from target sources.

How it works

The system crawls and collects data, then normalizes and organizes extracted information into usable formats. AI-assisted processing handles summarization, filtering, and structuring of results.
